# ukrainian-mods-translate
Репозиторій для проєкту перекладу модів модів на українську мову

Тут не будуть виходити файли оновлень, лише на [Planet Minecraft](https://www.planetminecraft.com/texture-pack/-6415904/) та [Modrint](https://modrinth.com/resourcepack/ukrainian-mods-translate).
Тут будуть лише файли перекладу і деякі інші файли.

Ви можете створювати запити на зміни(pull requests).
Вони можуть бути не прийняті, якщо мені не сподобаються зміни.
